## 🚀 How to Use

### 1. Start the Backend
```bash
cd c:\Users\Mega Store\Desktop\AroundU_Backend
python src/main.py
```

### 2. Open the App
Open: `frontend/user/index.html` in your browser

### 3. Flow
1. **Splash** (2 sec) → Auto-redirect
2. **Onboarding** (first time) → 3 screens
3. **Welcome** → Choose Login or Sign Up
4. **Login/Signup** → Enter credentials
5. **Places** → Browse places
6. Use **Bottom Nav** to switch between:
   - 🏠 Home (Places)
   - 🔍 Search
   - 💬 Chat
   - 👤 Profile
